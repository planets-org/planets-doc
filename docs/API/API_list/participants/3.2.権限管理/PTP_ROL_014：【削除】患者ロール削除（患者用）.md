### 処理概要

患者ロール紐付けを削除する。

| 機能 ID     | API 論理名                          | HTTP メソッド | URI                                              |
| :---------- | :---------------------------------- | :------------ | :----------------------------------------------- |
| PTP_ROL_014 | 【削除】本人以外ロール削除(患者用) | PUT          | {applicationPath}/participants/roles/assignment/{operatorTargetId} |

| 連携方式 | データ形式                           | 利用可能な接続先   |
| :------- | :----------------------------------- | :----------------- |
| REST API | JSON 形式（エンコーディング：utf-8） | ローカル、リモート |
