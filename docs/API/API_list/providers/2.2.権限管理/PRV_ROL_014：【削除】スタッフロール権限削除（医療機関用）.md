### 処理概要

スタッフロール紐付けを削除する。

| 機能 ID     | API 論理名                          | HTTP メソッド | URI                                              |
| :---------- | :---------------------------------- | :------------ | :----------------------------------------------- |
| PRV_ROL_014 | 【削除】スタッフロール権限削除(医療機関用) | PUT        | {applicationPath}/providers/roles/assignment |

| 連携方式 | データ形式                           | 利用可能な接続先   |
| :------- | :----------------------------------- | :----------------- |
| REST API | JSON 形式（エンコーディング：utf-8） | ローカル、リモート |
