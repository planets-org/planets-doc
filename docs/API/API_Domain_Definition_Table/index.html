APIパラメータ一覧

No.項目名属性設定要領
1値コードstringIDE：漢字、SYL：カナ　※認証情報には漢字が登録される
2演算子stringパスに対して値をどうチェックするか
01:＝ ※ 現時点では「＝」のみ
3救急フラグstring0:通常、1:救急
4権限種別string01:ReadOnly(参照のみ)
02:UpdateOnly(参照、更新、削除)
03:FullAccess(参照、登録、更新、削除)
04:AccessDeny(権限無し)
5権限承認状態string権限の承認状態を設定する。
0:承認要求中
1:承認済み
2:承認拒否
3:承認取下げ
6権限付与状態string権限の承認状態を設定する。
1:承認済み
2:承認拒否
7権限保持対象区分string権限保持対象区分を設定する。
1:個人
2:組織
8参照先（患者管理）string”self”/”remote”/"all"もしくは医療機関 ID のカンマ区切りを URL エンコードを行い指定
9参照先（権限管理）string”self”/”remote”/"all"
10参照先（組織管理）string"all"もしくは医療機関 ID のカンマ区切りを URL エンコードを行い指定
11参照先（スタッフ管理）string”self”/”remote”/"all"もしくは医療機関 ID のカンマ区切りを URL エンコードを行い指定
12参照先（文書管理）string”self”/”remote”/"all"もしくは医療機関 ID のカンマ区切りを URL エンコードを行い指定
13参照先（ユーザ管理）string”self”/”remote”/"all"もしくは医療機関 ID のカンマ区切りを URL エンコードを行い指定
14性別string"male" か "female"で検索
15論理削除フラグinteger権限の削除状態を設定する。
0:未削除
1:削除済み
